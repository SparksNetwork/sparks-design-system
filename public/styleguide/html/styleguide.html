Design tokens are the visual design atoms of the design system â€” specifically, they are named entities that store visual design attributes. Background Color

▼
Token Example
$color-background

Default background color for the whole app.
$sardine
$color-background-alt

Second default background color for the whole app.
$white
$color-background-alt-2

Used when further differentiation is needed because $color-background and $color-background-alt are both being used in close proximity.
$blueberry-white
$color-background-alt-inverse

Alternative background color for dark portions of the app.
$grape
$color-background-highlight-search

Background color for highlighting text search results.
$mustard
$color-background-scrollbar

The background color of an internal scrollbar.
$blueberry-steam
$color-background-scrollbar-track

The background color of an internal scrollbar track.
$blueberry-cadet
$color-background-selection

Used as the background color for selected text and elements.
$blueberry-cadet
$color-background-row-hover

Used as the background color for the hover state on rows or items on list-like components.
$blueberry-water
