title: Color Weak
---
This utility sets weak color to the body copy.

|  |  |
| --------------- | ----------------------- |
| **Class Name:** | `u-text-color--weak` |
| **Applied to:** | Any text |
| **Outcome:**    | Weak color of text |
| **Required:**   | No, optional element or modifier |
| **Comment:**    |  |
