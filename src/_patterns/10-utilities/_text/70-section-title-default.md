---
title: Section Title
---
Section titles are interactive titles that open and close sections, typically 
on a form.

|  |  |
| --------------- | ----------------------- |
| **Class Name:** | `u-section__title` |
| **Applied to:** |  |
| **Outcome:**    | Title of a section, can contain an interactive button to expand/collapse sub sections(s) |
| **Required:**   | Yes |
| **Comment:**    |  |
